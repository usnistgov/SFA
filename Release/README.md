## Download

Download the most recent version of SFA (SFA-5.nn.zip) above.  The README FIRST file and Release Notes are included in the zip file.

To download any file, left-click on the file name and then on the download icon (Download raw file) to the right.  A popup should appear in a few seconds to Open or Save the file.  You can also click on View Raw for the Release Notes spreadsheet.  Right clicking on the file name will not work.
