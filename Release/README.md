## Download

Download the most recent version of SFA (SFA-5.nn.zip) above.  Also in the directory are the README FIRST file, 
Release Notes, User Guide, sample STEP files, and old versions of SFA.

To download any file, left-click on the file name and then on the download icon (Download raw file) to the right.  
A popup should appear in a few seconds to Open or Save the file.  Right clicking on the file name will not work.
